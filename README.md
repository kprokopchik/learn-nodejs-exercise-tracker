# Exercise Tracker

This is basic Node.js study project.

### Prerequisites
* Node.js >= 18 

### Install dependencies
```shell
npm install
```

### Run dev server
```shell
npm run dev
```

### Check local server
http://localhost:3000

### Build project
* Compiles TypeScript and generates JS code in `./dist/` folder.
```shell
npm run build
```

### Start server
```shell
npm start
```
